## Notes

- Local LLM calls remain off by default and require `ALFARAHEEDI_LLM_BASE_URL`.
- LLM suggestions are full-text, manual-apply, and never eligible for `fix --safe`.
- No model weights are bundled in the repository, Docker image, or Windows package.
- Docker build context was reduced by excluding frontend and package build artifacts.
