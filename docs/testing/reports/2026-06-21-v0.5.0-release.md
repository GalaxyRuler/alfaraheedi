# v0.5.0 Release Verification

Date: 2026-06-21

## Scope

`v0.5.0` changes the primary Windows user surface from the legacy local web package to a Tauri desktop writing companion. The companion checks text selected in other apps through an explicit hotkey flow, keeps deterministic analysis local, and presents a compact review window for safe fixes and manual suggestions.

## Release Notes Draft

### Highlights

- Windows desktop companion for text selected in other apps.
- Global hotkey flow: select text, press `Ctrl+Alt+A`, review suggestions, copy corrected text, or replace the source selection.
- Shared Rust service path for CLI, API, and desktop companion.
- Narrow deterministic English suggestions for common typos and `you are do`.
- Narrow Arabic conversational greeting suggestion for `كيف حال ما اخبار`-style text.
- Desktop app starts as a GUI app without an empty console window.

### Limitations

- No live Grammarly-style underlines across all apps yet.
- Clipboard-mediated capture/apply depends on the source app allowing copy and paste.
- Local LLM setup remains advanced/manual; no model weights are bundled.
- The Arabic and English rule sets remain intentionally small and are not broad grammar engines.
