# Nahou v1.0 Acceptance Matrix

This matrix tracks public-safe release evidence for the v1.0 product contract.
Detailed logs and screenshots that could contain user text belong under ignored
private report directories, not in source control.

| Surface | Capture | Analyze | Apply | Privacy | Evidence file | Release blocker |
| --- | --- | --- | --- | --- | --- | --- |
| Notepad | Desktop hotkey captures selected text through UIA or clipboard fallback. | Deterministic engine runs locally. | Replace selection or copy corrected text. | No raw text in logs; clipboard restored when possible. | `docs/testing/desktop-v1.0-manual-qa.md` | Yes |
| Browser textarea | Desktop hotkey captures selected text; extension can analyze safe editable fields. | Local deterministic engine or loopback API. | Guarded replacement or copy corrected text. | No hosted processing; sensitive fields excluded best effort. | `docs/testing/desktop-v1.0-manual-qa.md`; browser manual QA report | Yes |
| Browser input | Desktop hotkey captures selected text; extension supports safe text-like input types. | Local deterministic engine or loopback API. | Guarded replacement or copy corrected text. | Password, token, payment, OTP, and hidden/sentinel fields excluded best effort. | Browser manual QA report | Yes |
| WhatsApp Web text box | Desktop hotkey or browser extension when active editor is safely editable. | Local deterministic engine or loopback API. | Guarded replacement or copy corrected text. | No raw text retained; sanitized failure messages. | Browser manual QA report | Yes |
| Word selected text | Desktop selected-text flow; Office add-in reads selected range where APIs permit. | Local deterministic engine or loopback API. | Guarded replacement only if selection still matches; copy fallback. | Public docs disclose Office selection processing. | Office manual sideload QA report | Yes |
| PowerPoint selected text | Desktop selected-text flow; Office add-in reads selected shape text where APIs permit. | Local deterministic engine or loopback API. | Guarded replacement only when Office APIs provide a safe range; copy fallback. | Public docs disclose unsupported selection states. | Office manual sideload QA report | Yes |
| Chrome extension textarea | Content script detects safe textarea fields. | Local loopback API only. | Span-validated replacement. | No telemetry, hosted writing service, or remote code. | Browser extension release report | Yes |
| Chrome extension contenteditable | Content script handles simple editable DOM ranges where offsets remain aligned. | Local loopback API only. | DOM range replacement only when source span still matches. | Hidden editor internals and sentinels excluded best effort. | Browser extension release report | Yes |
| Office Word task pane | Office.js selected range read. | Local loopback API only. | Guarded range replacement or copy fallback. | Public privacy/support URLs valid before submission. | Office manual sideload QA report | Yes |
| Office PowerPoint task pane | Office.js selected shape/text read where available. | Local loopback API only. | Guarded replacement or unsupported-state fallback. | Manifest claims match tested hosts. | Office manual sideload QA report | Yes |
| No selection | Desktop and integrations report no selected text. | No analysis request with empty user text. | No replacement. | No raw text logged. | Desktop/browser/Office manual QA reports | Yes |
| Large selection | Desktop refuses by default after warning. | Analyze only after explicit user action if enabled. | No automatic replacement. | Prevents oversized raw text from reports/logs. | Desktop manual QA report | Yes |
| Offline mode | Desktop deterministic rules work without network. | Local engine works; optional LLM unavailable unless local runtime is running. | Replace/copy remains local. | No hosted fallback. | Desktop manual QA report | Yes |
