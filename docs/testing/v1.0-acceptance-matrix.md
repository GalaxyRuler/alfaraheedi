# Nahou v1.0 Acceptance Matrix

This matrix tracks public-safe release evidence for the v1.0 desktop foundation
contract. Detailed logs and screenshots that could contain user text belong
under ignored private report directories, not in source control.

## Desktop Foundation Gates

These rows block the v1.0 desktop-foundation release.

| Surface | Capture | Analyze | Apply | Privacy | Evidence file | Release blocker |
| --- | --- | --- | --- | --- | --- | --- |
| Notepad selected text | Desktop hotkey captures selected text through UIA or clipboard fallback. | Deterministic engine runs locally. | Copy corrected text or replace selection when the guarded clipboard-paste path is available. | No raw text in logs; clipboard restored when possible. | `docs/testing/desktop-v1.0-manual-qa.md` | Yes |
| No selection or unsupported selection | Desktop reports no selected text or unsupported state. | No analysis request with empty user text. | No replacement. | No raw text logged. | `docs/testing/desktop-v1.0-manual-qa.md` | Yes |
| Privacy and diagnostics | Capture status reports method, source app, counts, and category flags only. | Public reports contain no raw selected text. | Clipboard restore warnings are visible when relevant. | Private QA artifacts stay ignored. | `docs/testing/desktop-v1.0-manual-qa.md` | Yes |

## Deferred Integration Gates

These rows are tracked because they matter for future public claims, browser-store
submission, or Office sideload readiness. They do not block the desktop
foundation release unless a release note explicitly includes the integration.

| Surface | Capture | Analyze | Apply | Privacy | Evidence file | Release blocker |
| --- | --- | --- | --- | --- | --- | --- |
| Browser textarea/input | Browser extension can analyze safe editable fields in local-ready fixtures. | Local loopback API only. | Guarded replacement or copy corrected text. | Sensitive fields excluded best effort. | Browser manual QA report | No, deferred store gate |
| Gmail, WhatsApp Web, Google Docs, rich web editors | Requires disposable live-editor QA. | Local deterministic engine or loopback API. | Guarded replacement only when spans remain aligned. | No raw text retained; sanitized failure messages. | Browser manual QA report | No, deferred live-editor gate |
| Chrome extension textarea/contenteditable | Content script detects supported editable fields. | Local loopback API only. | Span-validated replacement. | No telemetry, hosted writing service, or remote code. | Browser extension release report | No, local-ready only |
| Office Word task pane | Office.js selected range read requires live sideload QA. | Local loopback API only. | Guarded range replacement or copy fallback. | Public privacy/support URLs valid before submission. | Office manual sideload QA report | No, deferred sideload gate |
| Office PowerPoint task pane | Office.js selected shape/text read requires live sideload QA. | Local loopback API only. | Guarded replacement or unsupported-state fallback. | Manifest claims match tested hosts. | Office manual sideload QA report | No, deferred sideload gate |
| Large selection | Desktop should refuse by default after warning. | Analyze only after explicit user action if enabled. | No automatic replacement. | Prevents oversized raw text from reports/logs. | Desktop manual QA report | No, deferred hardening gate |
| Offline mode | Desktop deterministic rules work without network. | Optional LLM unavailable unless local runtime is running. | Copy/replace remains local. | No hosted fallback. | Desktop manual QA report | No, deferred hardening gate |
