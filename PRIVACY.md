# Privacy Policy

Nahou is local-first. The default CLI and local API analyze text in memory and do not retain submitted text.

The v0.7 browser extension has a separate store-facing privacy policy at
`browser-extension/PRIVACY_POLICY.md`. It covers active editable-field text,
loopback-only API access, stored extension settings, sensitive-field skipping,
and the no telemetry/no hosted writing service posture.

The project currently has no hosted service. If hosted mode is added, it must define retention, redaction, access controls, deletion, and audit behavior before launch.
